https://cordova.apache.org/docs/en/4.0.0/guide_cli_index.md.html

prerequisites:
Node.js for Command line Interface
Gradel for build automation

Steps:
1) Cordova -d create D:\Workspace\ProjectName AppId(optional) "DisplayName".
 
2)Move to the Directory of the project for adding the platform and plugings

For platform :

D:\Workspace\ProjectName> cordova -d platform add android/ios/windows

For Plugin :

D:\Workspace\ProjectName > cordova -d plugin add pluginname.

3)Prepare a project

D:\Workspace\ProjectName > cordova -d prepare

4) Build the project

D:\Workspace\ProjectName >cordova -d build android/ios
